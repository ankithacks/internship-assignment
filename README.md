# User Profile Management API

A RESTful API for user profile management with JWT authentication.

## Features

- User registration and login
- JWT authentication
- Protected routes
- User profile management
- Password hashing
- Error handling

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt.js
